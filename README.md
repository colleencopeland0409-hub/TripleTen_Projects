# TripleTen_Projects
Projects completed while at TripleTen in the Business Intelligence Bootcamp, and some personal projects for growth.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | P&L Evaluation (Sprint 4 Project) | The project was to look at profits and losses for a superstore, and determine which categories (or subcategories) of products were no longer beneficial to carry |
| 2 | Returns Rates and Trends for Superstore (Sprint 5 Project) | An interactive Tableau story analyzing retail return patterns to identify seasonal peaks, high-risk product categories like technology, and specific geographical trends in Southern California to drive data-backed prevention strategies. |
| 3 | Cycle Variablilty and Prediction Relability Case Study | This Tableau project analyzes user engagement and biological cycle patterns for a menstrual tracking application to improve prediction accuracy and provide data-driven recommendations for personalized user onboarding. |
| 4 | User Trust and Prediction Confidence in Menstrual Tracking | Project use to show return rates and trends for a superstore data set, with insights and A Tableau data story analyzing user cycle consistency and tracking history to develop an adaptive prediction model that enhances app reliability through personalized onboarding and targeted engagement strategies |
